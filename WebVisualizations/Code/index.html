Weather Data Analysis HomePage

The purpose of this project was to analyze how weather changes based on the latitude. To research the question further, I pulled data from the OpenWeatherMap API to generate a randomized list of over 500 cities. After assembling the data I used MatPlotLib to create multiple visualizations that help tell the story of the gathered data. Factors analyzed include Max. Temperature, Cloudiness, Humidity, and Wind Speed. This site has all of the visualizations, their unique analysis, and the raw data table.
